# Semana II: Lobees AI Automation Dashboard Challenge

En esta segunda parte de mis prácticas, me enfrento al reto de automatizar el procesamiento de tareas complejas en el CRM de Lobees utilizando agentes de IA avanzados. Mi objetivo es que Antigravity (y el resto de herramientas) actúen de forma autónoma gestionando datos, generando métricas y publicando un dashboard interactivo para el tutor.

## Mi enfoque para este reto

No me voy a limitar a que la automatización "funcione". Quiero que la experiencia de uso sea premium:
1. **Autonomía total**: El agente debe ser capaz de listar, procesar y finalizar tareas con un solo comando.
2. **Visualización de alto impacto**: El dashboard HTML que generaré tendrá un diseño moderno, responsive y alineado con la marca Lobees (#1A56A0).
3. **Trazabilidad clara**: Cada paso del proceso quedará registrado en los logs del CRM para una supervisión total.

## Estructura de mi solución

He organizado el repositorio siguiendo los requerimientos oficiales del reto para facilitar la evaluación:

- `CLAUDE.md`: Aquí defino mis Skills (instrucciones maestras).
- `antigravity/`: Mi configuración específica como agente de esta plataforma.
- `chatgpt/`: Preparación para el despliegue en GPTs personalizados.
- `dashboards/`: El histórico de los HTMLs generados durante las pruebas.
- `docs/`: Mi análisis comparativo sobre el rendimiento de cada IA.
